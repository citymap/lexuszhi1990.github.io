### Linux下*.tar.gz文件解压缩命令
1.压缩命令：

- 命令格式：tar  -zcvf   压缩文件名.tar.gz   被压缩文件名
- 可先切换到当前目录下。压缩文件名和被压缩文件名都可加入路径。


2.解压缩命令：

- 命令格式：tar  -zxvf   压缩文件名.tar.gz
- 解压缩后的文件只能放在当前的目录。

`-v` 对每一个压缩和解压的文件，显示文件名和压缩比。

### linux下rar包的压缩与解压方案 

1.压缩功能

`sudo apt-get install rar`

`unrar e update.rar`

2.解压功能

`sudo apt-get install unrar`

`unrar x update.rar`
